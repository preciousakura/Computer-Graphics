## Compile and Run

* Please, run in any linux distribution

* Compile
```bash
    make
```

* Execute 
```bash
    ./main
```

# Alternative Compile
```
g++ main.cpp implementation/*.cpp -Ofast -flto -pipe -lSDL2 -o main && ./main
```
